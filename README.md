
# C# and .NET Framework – Part 1

Course Overview & Learning Outcomes

Welcome! This course introduces the fundamentals of C# programming and the .NET Framework. By the end, you will have a solid foundation in object-oriented programming (OOP), core C# syntax, and the structure of .NET applications.


## 📘 What You Will Learn
- 1. Programming Fundamentals & C# Basics
Define fundamental programming terms related to C#

Describe object-oriented programming (OOP) concepts

Explain how the .NET Framework accommodates OOP

- 2. .NET Architecture & Tools
Identify the stages of the .NET Common Language Infrastructure (CLI)

Explain how to use the C# compiler (csc)

Explain how to write a C# program in Visual Studio

- 3. Data, Variables & Type Management
Describe various data types used with C# (int, double, string, bool, etc.)

Explain how to use casting (implicit/explicit conversion)

Explain how to assign variables in C#

Describe how to execute/manipulate data types

- 4. Operations & Logic
Explain how to perform math operations (+, -, *, /, %)

Explain Boolean logic (&&, ||, !, comparison operators)

Describe how to use if/else statements

- 5. Control Flow & Repeated Execution
Explain branching (switch, if-else if-else)

Explain how to design, write, and execute loops (for, while, do-while)

Work with strings (concatenation, length, indexing, basic methods)

## 🛠️ Tools You’ll Use
.NET SDK (includes C# compiler)

Visual Studio (Community edition or higher)

C# programming language

## ✅ Recommended Practice After Each Topic
